Instructions to run, type the following commands into your terminal:

Clone repository: git clone https://github.com/FarCry88/SeaofThievesRandomizer.git
Navigate to cloned folder: cd SeaofThievesRandomizer
Ensure Docker Desktop is installed and running (IMPORTANT).
Build and run the docker file using this command in the terminal: docker compose up
Once initialized, navigate to your browser and enter: https://localhost
Web page should now display

Click get event to recieve your randomized event
